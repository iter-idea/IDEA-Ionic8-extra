# IDEARCPickerComponent

## Selector

idea-rc-picker

## Inputs

- `team` (_string_) - The team from which we want to load the resources. Default: try to guess current team.
- `folder` (_RCConfiguredFolder_) - The folder of which to load the resources.
- `attachedResources` (_RCAttachedResource[]_) - The array in which we want to add/remove resources.
- `editMode` (_boolean_) - Regulate the mode (view/edit).
- `lines` (_string_) - The lines attribute of the item.
