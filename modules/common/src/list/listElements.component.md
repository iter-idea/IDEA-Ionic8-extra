# IDEAListElementsComponent

## Selector

idea-list-elements

## Inputs

- `data` (_(string | Label)[]_) - It should be read only until the component closure.
- `labelElements` (_boolean_) - Whether the elements are labels or simple strings.
- `searchPlaceholder` (_string_) - A placeholder for the searchbar.
- `noElementsFoundText` (_string_) - The text to show in case no element is found after a search.
