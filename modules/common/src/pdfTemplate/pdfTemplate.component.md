# IDEAPDFTemplateComponent

## Selector

idea-pdf-template

## Inputs

- `blueprint` (_PDFTemplateBlueprint_) - The blueprint to define the allowed sections and fields for this PDF template.
- `template` (_PDFTemplateSection[]_) - The PDF template to manage, as list of sections.
- `languages` (_Languages_) - Language preferences.
- `disabled` (_boolean_) - If true, the component is disabled.
