# IDEAShowHintButtonComponent

## Selector

idea-show-hint-button

## Inputs

- `hint` (_string_) - The string to show as title of the alert.
- `message` (_string_) - The string to show as content of the alert.
  If not specified, it's equal to `hint.concat('_I')`.
- `confirmationText` (_string_) - The string to show as the alert's confirmation button.
- `cssClass` (_string_) - A CSS class to apply to the alert box.
- `slot` (_string_) - The slots where to put the button.
- `fill` (_string_) - The fill attribute of the button.
- `color` (_string_) - The color of the button.
- `icon` (_string_) - The IonIcon to show for the button.
