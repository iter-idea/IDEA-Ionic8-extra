# IDEAAddressComponent

## Selector

idea-address

## Inputs

- `address` (_Address_) - The address to manage.
- `showContact` (_boolean_) - If true, show the field `contact`.
- `showAddress2` (_boolean_) - If true, show the field `address2`.
- `showPhone` (_boolean_) - If true, show the field `phone`.
- `showEmail` (_boolean_) - If true, show the field `email`.
- `editMode` (_boolean_) - Whether the fields are editable or disabled.
- `obligatory` (_boolean_) - If true, show obligatory dots.
- `lines` (_string_) - The lines attribute of the item.
- `color` (_string_) - The color for the component.
- `label` (_string_) - The label to show for the field; if not set, it has a default value.
- `placeholder` (_string_) - The placeholder to show for the field.
- `openByDefault` (_boolean_) - To toggle the detailed view.

## Outputs

- `addressChange` (_Address_)
