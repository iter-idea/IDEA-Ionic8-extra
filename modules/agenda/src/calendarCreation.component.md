# IDEACalendarCreationComponent

## Selector

idea-calendar-creation

## Inputs

- `fixScope` (_CalendarScopes_) - Whether we want to allow the creation of only a particular type of calendar, based on the scope.
- `onlyExternalCalendars` (_boolean_) - Whether we want to allow the creation of local calendars.
- `baseURL` (_string_) - The URL to be used on the redirect calls.
